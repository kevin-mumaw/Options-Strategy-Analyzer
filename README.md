# 📊 Options Strategy Analyzer

A Google Colab-based options strategy research notebook for identifying and validating directional options trades using technical analysis, reversal detection, market regime filtering, and historical backtesting.

## Project Status
- **Current Development Version:** v2.6
- **Status:** Active development and validation
- **Platform:** Google Colab
- **Language:** Python

## Core Capabilities
- Technical indicator analysis
- Reversal pattern detection
- Multi-timeframe confirmation
- Market regime filtering
- Options chain review
- Greeks display
- Position sizing and risk controls
- Historical backtesting
- CSV trade journaling

## Quick Start

### Open the Notebook
Use the notebook in the `notebooks/` folder as the current working version.

### Basic Workflow
1. Open the notebook in Google Colab
2. Run the setup and configuration cells
3. Choose **Live Scan** or **Backtest** mode
4. Review signals, options data, and risk metrics
5. Validate any strategy changes with backtesting before using them live
